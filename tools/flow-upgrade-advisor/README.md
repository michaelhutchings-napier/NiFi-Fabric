# Flow Upgrade Advisor Pointer

NiFi-Fabric does not own the Flow Upgrade Advisor implementation directly.

Use the standalone repository for offline Apache NiFi flow upgrade analysis,
deterministic rewrites, target validation, and publish workflows:

- `https://github.com/michaelhutchings-napier/nifi-flow-upgrade-advisor`

This directory stays intentionally small and acts only as a pointer for
NiFi-Fabric users who want the external tool.
